# vscode-pets.en

## Commands

| Command                        | Title                                            |
| ------------------------------ | ------------------------------------------------ |
| `vscode-pets.start`            | Pet Coding: Start pet coding session            |
| `vscode-pets.throw-with-mouse` | Pet Coding: Throw ball with mouse |
| `vscode-pets.throw-ball`       | Pet Coding: Throw ball       |
| `vscode-pets.export-pet-list`  | Pet Coding: Export pet list  |
| `vscode-pets.import-pet-list`  | Pet Coding: Import pet list  |
| `vscode-pets.spawn-pet`        | Pet Coding: Spawn pet        |
| `vscode-pets.delete-pet`       | Pet Coding: Delete pet       |
| `vscode-pets.remove-all-pets`  | Pet Coding: Remove all pets  |
| `vscode-pets.roll-call`        | Pet Coding: Roll call        |

## Configuration

| Key                              | Description                                  | Type      | Default      |
| -------------------------------- | -------------------------------------------- | --------- | ------------ |
| `vscode-pets.petColor`           | Color of the pet           | `string`  | `"brown"`    |
| `vscode-pets.petType`            | Type of the pet            | `string`  | `"cat"`      |
| `vscode-pets.petSize`            | Size of the pet            | `string`  | `"nano"`     |
| `vscode-pets.position`           | Position of the pets panel           | `string`  | `"explorer"` |
| `vscode-pets.theme`              | Theme              | `string`  | `"none"`     |
| `vscode-pets.throwBallWithMouse` | Throw ball with mouse | `boolean` | `false`      |
| `vscode-pets.disableEffects`     | Disable effects     | `boolean` | `false`      |
