## Commands

| Command                        | Title                                            |
| ------------------------------ | ------------------------------------------------ |
| `vscode-pets.start`            | Pet Coding: 启动宠物编程会话            |
| `vscode-pets.throw-with-mouse` | Pet Coding: 使用鼠标投掷球 |
| `vscode-pets.throw-ball`       | Pet Coding: 投掷球       |
| `vscode-pets.export-pet-list`  | Pet Coding: 导出宠物列表  |
| `vscode-pets.import-pet-list`  | Pet Coding: 导入宠物列表  |
| `vscode-pets.spawn-pet`        | Pet Coding: 创建宠物        |
| `vscode-pets.delete-pet`       | Pet Coding: 删除宠物       |
| `vscode-pets.remove-all-pets`  | Pet Coding: 移除全部宠物  |
| `vscode-pets.roll-call`        | Pet Coding: 点名        |

## Configuration

| Key                              | Description                                  | Type      | Default      |
| -------------------------------- | -------------------------------------------- | --------- | ------------ |
| `vscode-pets.petColor`           | 宠物颜色           | `string`  | `"brown"`    |
| `vscode-pets.petType`            | 宠物类型            | `string`  | `"cat"`      |
| `vscode-pets.petSize`            | 宠物大小            | `string`  | `"nano"`     |
| `vscode-pets.position`           | 宠物面板位置           | `string`  | `"explorer"` |
| `vscode-pets.theme`              | 主题              | `string`  | `"none"`     |
| `vscode-pets.throwBallWithMouse` | 使用鼠标投掷球 | `boolean` | `false`      |
| `vscode-pets.disableEffects`     | 禁用效果     | `boolean` | `false`      |
