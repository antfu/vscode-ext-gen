# vscode-pets.ja

## Commands

| Command                        | Title                                            |
| ------------------------------ | ------------------------------------------------ |
| `vscode-pets.start`            | Pet Coding: ペットコーディングセッションを開始            |
| `vscode-pets.throw-with-mouse` | Pet Coding: マウスでボールを投げる |
| `vscode-pets.throw-ball`       | Pet Coding: ボールを投げる       |
| `vscode-pets.export-pet-list`  | Pet Coding: ペットリストをエクスポート  |
| `vscode-pets.import-pet-list`  | Pet Coding: ペットリストをインポート  |
| `vscode-pets.spawn-pet`        | Pet Coding: ペットを生成        |
| `vscode-pets.delete-pet`       | Pet Coding: ペットを削除       |
| `vscode-pets.remove-all-pets`  | Pet Coding: すべてのペットを削除  |
| `vscode-pets.roll-call`        | Pet Coding: 点呼        |

## Configuration

| Key                              | Description                                  | Type      | Default      |
| -------------------------------- | -------------------------------------------- | --------- | ------------ |
| `vscode-pets.petColor`           | ペットの色           | `string`  | `"brown"`    |
| `vscode-pets.petType`            | ペットの種類            | `string`  | `"cat"`      |
| `vscode-pets.petSize`            | ペットのサイズ            | `string`  | `"nano"`     |
| `vscode-pets.position`           | ペットパネルの位置           | `string`  | `"explorer"` |
| `vscode-pets.theme`              | テーマ              | `string`  | `"none"`     |
| `vscode-pets.throwBallWithMouse` | マウスでボールを投げる | `boolean` | `false`      |
| `vscode-pets.disableEffects`     | エフェクトを無効にする     | `boolean` | `false`      |
